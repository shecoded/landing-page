# AprendeOnline

Landing page de una plataforma de formación online diseñada para presentar cursos y contenido educativo de forma clara, organizada y visual.

## 📌 ¿Qué es?

Este proyecto es una página web tipo landing que simula una plataforma de aprendizaje online. Su objetivo es mostrar información relevante (cursos, testimonios y contacto) de manera estructurada y accesible para el usuario.

## 🚀 Funcionalidades

- Sección de presentación (hero)
- Listado de cursos
- Testimonios de usuarios
- Preguntas frecuentes (FAQ)
- Formulario de contacto
- Navegación entre secciones

## 🛠 Tecnologías

- HTML
- CSS

## 📱 Responsive

- Diseño adaptable básico
- Uso de Flexbox y Grid para estructura
- Media query para adaptar la sección de cursos en pantallas más grandes

## 📂 Estructura del proyecto

- index.html
- style.css

## 📚 Aprendizajes

- Uso de HTML semántico para estructurar una página web
- Organización de contenido en secciones
- Uso de Flexbox y Grid
- Creación de layouts responsive básicos
- Mejora en la maquetación y diseño visual

## 🔧 Mejoras futuras
